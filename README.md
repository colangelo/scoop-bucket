# Scoop Bucket

Scoop bucket for [hp (HittyPing)](https://github.com/colangelo/HittyPing) - a prettyping-style HTTP(S) latency monitor.

## Installation

```powershell
scoop bucket add colangelo https://github.com/colangelo/scoop-bucket
scoop install hp
```

## Update

```powershell
scoop update hp
```

## Available Apps

| App | Description |
|-----|-------------|
| [hp](https://github.com/colangelo/HittyPing) | Prettyping-style HTTP(S) latency monitor |
