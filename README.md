# Scoop Bucket

Scoop bucket for hp and other tools

## Installation

```powershell
scoop bucket add colangelo https://github.com/colangelo/scoop-bucket
scoop install hp
```

## Update

```powershell
scoop update hp
```

## Available Apps

| App | Description |
|-----|-------------|
| [hp](https://github.com/colangelo/HittyPing) | Prettyping-style HTTP(S) latency monitor |
